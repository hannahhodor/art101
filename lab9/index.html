Challenge

The challenge of this lab is to begin using JavaScript to edit HTML elements that already exist as well as creating new HTML elements through JavaScript.

Problems

We misread the instructions on the Canvas page, causing us to think that we needed to append both of the new elements at once. We learned that we could use .append instead of .appendChild to do so. The latter method only appends one child at a time. We also had problems getting our button to work. It seemed like if we created the button event listener [the thing that senses if a button is clicked] through JS instead of the CSS/HTML, the function of the button would run when the page was opened rather than when the button was clicked. We also occasionally forgot to add the .document to JS methods that would affect elements that already exist in the HTML.
